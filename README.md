# 진화형 AI 코드 메이커

사용자가 원하는 앱 목표를 입력하면 기본 코드를 생성하고, 문법 검사 결과와 성공/실패 기록을 저장해서 다음 코드 생성에 반영하는 Streamlit 앱입니다.

## 핵심 기능

- 목표 입력 후 코드 프로젝트 생성
- Streamlit 기본 앱 / API 대시보드 자동 분류
- ai_memory.json에 학습 기록 저장
- 성공/실패 패턴 저장
- 다음 개선용 프롬프트 자동 생성
- 생성 프로젝트 ZIP 다운로드
- 모바일 화면에서 보기 쉽게 큰 UI 적용

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 중요한 설명

이 앱은 위험하게 자기 코드를 마음대로 바꾸는 AI가 아닙니다.
대신 안전하게 아래 방식으로 진화합니다.

1. 사용자가 목표를 입력한다.
2. 템플릿 기반으로 코드를 생성한다.
3. 문법 검사를 한다.
4. 성공/실패 결과를 기억한다.
5. 다음 생성 때 기억을 프롬프트와 패턴으로 반영한다.
